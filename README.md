# TMFShark SDK

TMFShark是一个高性能的网络框架。

## Swift Package Manager

### 安装

在Xcode中，选择 File > Add Package Dependencies，然后输入：

```
https://github.com/your-username/TMFShark.git
```

### 使用

```swift
import TMFShark

// 使用TMFShark SDK的功能
```

## 要求

- iOS 11.0+
- Xcode 12.0+
- Swift 5.5+
